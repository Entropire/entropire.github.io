SimpleCommands

A lightweight command interface for fast prototyping in C# with commands

  • Table of Contents

Description

I built this API to speed up my workflow when making small CLI tools or prototypes. It it allows registering commands via class inharitance or lambda expressions. It also makes it easy to switch between command handlers so you can change active command easily.

Features

  • Command registration and execution system
  • Support for commands via class inheritance or lambda expressions
  • Command handler system for managing multiple sets of commands
  • Easy execution of commands by name via a unified interface

Commands

A command is an order given to a computer program that performs a specific action via a command-line interface (CLI). Commands can be created with one of two ways.

  1. By extending from the Command class:
class GreetCommand : Command
{
    public override void Execute(string[] args)
    {
        Console.WriteLine($&quot;Hello, {args[0]}!&quot;);
    }
}

//Creates a class that inherits from **Command** and prints `Hello, [the user's first arg]!` to the console when executed.

  1. By using a lambda expression:
handler.Register(&quot;greet&quot;, args =&gt; {
    Console.WriteLine($&quot;Hello, {args[0]}!&quot;); 
});

//Creates a command lambda expression that prints `Hello, [the user's first arg]!` to the console when executed.

Command Name

Command names are the names that you enter in a command line interface or a console to execute the code linked to that name. There are two ways to link a command to a name:

  1. By adding the CommandName attribute to your command class:
[CommandName(&quot;greet&quot;)]
class GreetCommand : Command
{
    public override void Execute(string[] args)
    {
        Console.WriteLine($&quot;Hello, {args[0]}!&quot;);
    }
}

//Creates a class that inherits from **Command** where the command name is set with the **CommandName** attribute.

  1. By setting the command name while registering your command:
handler.Register(&quot;greet&quot;, new GreetCommand());

//Registers a command where the name of the command is set as a parameter of the **Register** function.

CommandHandler

A command handler is a registry that manages all registered commands and routes execution to the correct one. You can create a new command handler by instantiating a new one:

CommandHandler handler = new CommandHandler();

//Creates a new **CommandHandler** instance.

To register commands call the **Register** function:

handler.Register(new GreetCommand());

//Registers the **GreetCommand** where the name is set in the class with the **CommandName** attribute.

To unregister commands call the **Unregister** function:

handler.Unregister(new GreetCommand());

//Unregisters the **GreetCommand**.

To execute a command call the **Execute** function:

handler.Execute(&quot;greet&quot;, &quot;helloworld&quot;);

//Executes the command linked to the `greet` keyword.
